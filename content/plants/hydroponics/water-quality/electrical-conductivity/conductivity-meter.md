##### How to Set ATC (Automatic Temperature Calibration)
- Power the device on.
- Hold down the power button for 2 seconds to enter setup mode.
- Use the up and down arrows to select the section slug option.
- Use the up and down arrows to select the sub-section # option.
- Use the up and down arrows to select set the option's value.
- Press ESC a couple of times to go back to home.
