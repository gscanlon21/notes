# Graphing Calculator

## Table Functions
##### How to Add a Line Between Points in a Table
Long click on the table header button (looks like dots) to change how the points are displayed.

## Labels
To add a label, click on the graph to add a point `(x,y)`. A label icon should appear under the point.
