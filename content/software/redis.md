# Redis
An in-memory data structure store. 

## [Install Redis](https://redis.io/download)

### Via scoop
```powershell
scoop install redis
```

## Start the Redis server

```powershell
redis-server
```

### Check the server is running

```powershell
redis-cli ping
```
