chkdsk

Checks the file system and file system metadata of a volume for logical and physical errors. If used without parameters, chkdsk displays only the status of the volume and does not fix any errors. If used with the /f, /r, /x, or /b parameters, it fixes errors on the volume.[1]

Fix Corrupted Files or Folders

sudo chkdsk /I /C /F

And then restart your computer.
